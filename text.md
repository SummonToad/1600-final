# Binary to Text: Converting Binary to Characters

Computers store text as numbers using **ASCII** or **Unicode**. Each character corresponds to a binary number.

### ASCII Example
| Character | ASCII Decimal | Binary      |
|-----------|---------------|------------|
| A         | 65            | 01000001   |
| B         | 66            | 01000010   |
| H         | 72            | 01001000   |
| i         | 105           | 01101001   |

### Converting Binary to Text
1. Take each 8-bit binary number (1 byte).  
2. Convert it to decimal.  
3. Look up the corresponding ASCII character.
